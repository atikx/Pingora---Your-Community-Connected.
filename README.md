# 🚀 Pingora — Your Community Connected

## Getting Started

Follow these steps to run the project locally:

### 1. Start the Frontend
```bash
cd ./client
npm install
npm run dev
```


### 2. Start the Backend

cd ./server
npm install
node index.js



## Prerequisites

Make sure you have the following installed:
- Node.js (v16 or higher)
- npm 
- Git
