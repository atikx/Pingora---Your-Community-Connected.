## Installation

After clonning Open Terminal and run following command

```bash
  cd ./client
  npm i
  npm run dev
```

Open new Terminal and run the following commands

```bash
  cd ./server
  npm i
  npm run dev
```

## Prerequisites

Make sure you have the following installed:
- Node.js (v16 or higher)
- npm 
- Git
